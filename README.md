# Прикиньте будущую зарплату

Скрипт анализирует вакансии на Habr и SuperJob по 8 популярным языкам программирования: JavaScript, Java, Python, Ruby, PHP, C++, CSS, C#.

### Как установить

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
Для запуска скрипта клонируйте репозиторий или скачайте скрипт.
Зарегистрируйтесь на сайте и получите свой API-токен сервиса по ссылке [API SuperJob](https://api.superjob.ru/).

Для сокрытия чувствительных данных, таких как API-токен сервиса, сохраните их в папке со скриптом в отдельном `.env` файле в формате `SUPERJOB_API_KEY=ваш_api_токен`.

### Пример запуска 

Запуск осуществляется командой в терминале `python salary.py`. В результате отображается таблица с данными по вакансиям программистов в Москве:
